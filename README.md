# Personal AI Agent Skills

Custom skills for AI coding agents (Claude Code, OpenCode, Copilot, Gemini, etc.)

## Skills

### git-workflow

Git workflow automation and best practices for AI agents.

## Usage

Install all skills from this repo:

```bash
bunx skills add abijith-suresh/skills
```

Or install individual skills by referencing the specific skill folder.

## Adding New Skills

1. Create a new folder: `mkdir my-skill/`
2. Add `SKILL.md` with the skill definition
3. Commit and push
4. Update your dotfiles script to include the new skill
